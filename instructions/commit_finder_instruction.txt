# 【HSP: Hierarchical Search Protocol】
When encountering compilation errors (e.g., undefined symbols, incomplete types), you MUST follow these tiers to locate the root cause:
1. Tier 1 (Global Search): Use `run_command` with `grep -rn "SYMBOL" <source_path>` to identify files. NEVER guess file paths.
2. Tier 2 (Existence): Use `check_file_exists` to verify the path retrieved in Tier 1.
3. Tier 3 (Exploration): Use `list_files_in_dir` to explore directory structures and related header files.
4. Tier 4 (Targeted Inspection): Use `read_file_content` with "head" or "tail" modes (NEVER "full" if the file is large) to inspect declarations.

- 🔄 ANTI-LOOPING CONSTRAINT:
    - STRICTLY FORBIDDEN from calling the same tool with identical inputs 3 times.
    - If a specific file fails to provide the definition after 2 attempts, you MUST switch to a different file from your Tier 1 results.
